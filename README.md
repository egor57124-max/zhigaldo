## 📱 **О игре**

«Воздушный хоккей: Три кота» — это аркадная игра, где вы управляете ракеткой с котятами и соревнуетесь с компьютерным соперником Коржиком. Игра полностью оптимизирована для мобильных устройств и отлично работает на всех современных браузерах.

### 🎯 **Цель игры**
Первым забить **7 голов** в ворота соперника!

### 🎮 **Управление**
| Устройство | Управление |
|------------|------------|
| 📱 **Телефон/Планшет** | Проведите пальцем по экрану, чтобы двигать ракетку |
| 🖱️ **Компьютер** | Двигайте мышью по игровому полю |

## 🛠 **Технологии**

Проект создан на чистом **HTML/CSS/JavaScript** без использования сторонних библиотек:

- **HTML5 Canvas** — отрисовка графики
- **Vanilla JavaScript** — игровая логика
- **CSS3** — адаптивный дизайн и анимации
- **Web Audio API** — звуковые эффекты

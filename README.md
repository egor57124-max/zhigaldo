# 🏒 Воздушный хоккей: Три кота

Простая и весёлая игра в воздушный хоккей с персонажами мультсериала «Три кота».

---

## 🚀 **Как запустить игру**

### Способ 1: Просто открыть файл
1. Сохраните код выше как `index.html`
2. Дважды кликните по файлу
3. Игра откроется в браузере!

### Способ 2: Если не работает
- Используйте **Google Chrome** или **Safari**
- На телефоне: скачайте файл и откройте в браузере
- Убедитесь, что файл имеет расширение `.html`

### Способ 3: Быстрая проверка
Скопируйте код и вставьте на [codepen.io](https://codepen.io/pen/)

---

## 🎮 **Как играть**

| Действие | Как сделать |
|----------|-------------|
| Двигать ракетку | Ведите пальцем (телефон) или мышкой (компьютер) |
| Забить гол | Попасть шайбой в верхние ворота |
| Победить | Набрать 7 очков |
